# Customer Churn Prediction

Predicts customer churn in the travel industry using machine learning. Includes end-to-end analysis, a trained model, and a web app for real-time predictions.

## Setup

```bash
git clone https://github.com/dhvanitpadamshali/customer-churn-prediction.git
cd customer-churn-prediction
pip install -r requirements.txt
python app.py
```

Then open `http://localhost:5000` in your browser.

## Project Structure

| File | Description |
|---|---|
| `Customer_Churn_Prediction.ipynb` | EDA, feature engineering, model training |
| `app.py` | Web app for real-time predictions |
| `model.pkl` | Serialized trained model |
| `customer_travel_churn.csv` | Dataset |

## Stack

Python · pandas · scikit-learn · Matplotlib · Flask
